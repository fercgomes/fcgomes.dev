# Media Directory

This directory contains media files for showcasing your work.

## Directory Structure

- `images/` - Screenshots, mockups, UI demos
- `videos/` - Video demonstrations of features
- `charts/` - Charts, graphs, analytics visualizations

## Usage

1. Add your media files to the appropriate subdirectory
2. Update `components/media-showcase.tsx` with your media items
3. Use the following path format:
   - Images: `/media/images/your-file.png`
   - Videos: `/media/videos/your-file.mp4`
   - Charts: `/media/charts/your-file.png`

## File Recommendations

### Images
- Format: PNG, JPG, or WebP
- Recommended: WebP for best compression
- Max width: 1920px for optimal display

### Videos
- Format: MP4 (H.264 codec)
- Recommended: Keep file sizes reasonable (< 10MB if possible)
- Consider using video hosting (YouTube, Vimeo) for larger files and embed instead

### Charts
- Format: PNG, SVG, or WebP
- SVG recommended for scalability
